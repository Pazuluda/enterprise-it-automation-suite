## Politique de modification

EITAS Identity privilégie :

1. les thèmes personnalisés ;
2. les fournisseurs SPI ;
3. les configurations ;
4. les scripts de construction ;
5. les modifications minimales et documentées du cœur.

La marque EITAS Identity désigne la distribution personnalisée.
Elle ne doit pas être présentée comme un projet officiel publié par
l’équipe Keycloak.
